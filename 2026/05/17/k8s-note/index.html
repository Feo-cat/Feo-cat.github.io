Kubernetes（K8s）入门笔记

Kubernetes（简称 K8s）是目前最主流的容器编排平台，主要用于管理 Docker 容器，实现自动部署、扩容、服务发现和高可用。

Kubernetes 是什么

Kubernetes 是 Google 开源的容器编排系统，用于自动化部署、扩展和管理容器化应用。

简单来说：

  • Docker 负责“运行容器”
  • Kubernetes 负责“管理容器”

Pod 是什么

Pod 是 Kubernetes 中最小的部署单位。
