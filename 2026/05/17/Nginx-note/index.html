Nginx 入门笔记

发表于2026-05-17|更新于2026-05-17|技术笔记
|浏览量:

Nginx 是一款高性能的 Web 服务器，也常被用作反向代理、负载均衡和静态资源服务器。它占用资源少、并发能力强，是后端部署中非常常见的基础组件。

Nginx 是什么

Nginx 是一个开源的 HTTP 服务器软件。和传统 Web 服务器相比，Nginx 更擅长处理高并发请求，因此经常用于网站入口层。

常见用途包括：

  • 部署静态网站
  • 反向代理后端服务
  • 配置 HTTPS
  • 做负载均衡
  • 缓存静态资源
  • 控制访问权限

常见应用场景

静态资源服务器

Nginx 可以直接托管 HTML、CSS、JavaScript、图片等静态文件。

  • listen 80 表示监听 80 端口

反向代理

反向代理是 Nginx 最常用的功能之一。用户访问 Nginx，Nginx 再把请求转发给后端服务。

这样用户访问 api.example.com 时，请求会被转发到本机的 3000 端口。

负载均衡

当后端服务有多个实例时，可以使用 Nginx 做简单的负载均衡。

Nginx 会把请求分发到不同的后端服务上，提高系统的可用性和并发能力。

小结

Nginx 是后端部署中非常重要的工具。它不仅可以作为 Web 服务器，还可以承担反向代理、负载均衡、HTTPS 配置等职责。

对于初学者来说，可以先掌握以下几个重点：

  • server 用来定义一个站点
  • location 用来匹配请求路径
  • root 用来指定静态资源目录
  • proxy_pass 用来转发请求
  • 修改配置后先执行 nginx -t
  • 配置无误后再执行 systemctl reload nginx

掌握这些基础内容后，就可以完成大多数简单的网站部署和后端代理配置。
